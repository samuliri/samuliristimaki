# [samuliristimaki.com](https://samuliristimaki.com)

## Features
- [Gatsby](https://www.gatsbyjs.org/)
- [React](https://reactjs.org/)
- [GraphQL](http://graphql.org/)
- [Firebase](https://firebase.google.com/)

## Install dependencies
```
yarn install
```

## Development
```
yarn develop
```

## Build
```
yarn build
```

## Deploy
```
firebase deploy
```
